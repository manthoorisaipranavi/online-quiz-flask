# Online Quiz System (Flask)

A basic role-based online quiz application built using **Python Flask**, where an **Admin** can create quiz questions and a **User** can attempt the quiz and view the score.

---

## Features

- Role selection: Admin / User
- Admin can:
  - Add quiz questions
  - Provide multiple options and correct answers
- User can:
  - Attempt the quiz
  - View final score after submission
- Automatic evaluation of answers
- Clean and simple user interface

---

## Tech Stack

- Python
- Flask
- HTML
- CSS
